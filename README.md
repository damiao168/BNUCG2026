

# 粒子群算法（PSO）可视化项目

## 项目简介

本项目实现了粒子群优化算法（Particle Swarm Optimization，PSO）的核心逻辑，并结合可视化技术呈现粒子群迭代寻优的交互效果。通过本项目，用户可以直观观察算法的寻优过程与收敛特性，深入理解粒子群算法的工作原理。

粒子群算法是一种基于群体智能的优化算法，模拟鸟群觅食行为，通过粒子之间的协作与信息共享来寻找全局最优解。该算法具有实现简单、收敛速度快、参数设置灵活等优点，广泛应用于函数优化、神经网络训练、路径规划等领域。

## 核心逻辑

### 1. 算法层（pso_core.py）

算法层是整个项目的核心，实现了粒子群优化的基本原理：

- **粒子属性定义**：每个粒子具有位置（position）和速度（velocity）两个基本属性
- **适应度计算**：根据目标函数计算每个粒子的适应度值，评估解的优劣
- **速度更新公式**：$v_{i}(t+1) = w \cdot v_{i}(t) + c_1 \cdot r_1 \cdot (pbest_i - x_i(t)) + c_2 \cdot r_2 \cdot (gbest - x_i(t))$
- **位置更新公式**：$x_{i}(t+1) = x_{i}(t) + v_{i}(t+1)$
- **全局最优更新**：通过迭代比较，实时更新全局最优位置

其中：
- $w$ 为惯性权重，控制粒子对之前速度的继承程度
- $c_1$ 为学习因子（个体认知），引导粒子向自身最优位置移动
- $c_2$ 为学习因子（社会认知），引导粒子向全局最优位置移动
- $r_1, r_2$ 为 $[0,1]$ 范围内的随机数

### 2. 可视化层（visualizer.py）

可视化层负责将算法运行过程以图形化方式呈现：

- **粒子分布绘制**：实时展示所有粒子在解空间中的位置
- **最优解轨迹**：绘制全局最优位置随迭代次数的变化曲线
- **交互控制**：支持动态调整算法参数，包括粒子数量、迭代次数、惯性权重、学习因子等
- **实时反馈**：在可视化界面上显示当前最优适应度值

### 3. 执行层（run.py）

执行层作为项目的入口点，负责整合各模块：

- 参数初始化配置
- 算法与可视化的协调运行
- 最终结果的输出与展示

## 实现功能

本项目实现了以下核心功能：

1. **实时可视化展示**：完整呈现粒子群算法寻优过程的动态效果，观察粒子在解空间中的搜索行为

2. **自定义目标函数支持**：
   - 单峰函数（如Sphere函数、Rosenbrock函数）
   - 多峰函数（如Rastrigin函数、Ackley函数）
   - 用户可根据需要替换目标函数进行测试

3. **参数可调特性**：
   - 粒子数量：控制种群规模
   - 迭代次数：控制算法运行轮数
   - 惯性权重 $w$：影响全局搜索与局部搜索的平衡
   - 学习因子 $c_1, c_2$：影响粒子向个体最优和全局最优学习的程度

4. **结果输出**：程序结束后输出最终寻优结果，包括：
   - 全局最优位置坐标
   - 最优适应度值
   - 收敛曲线数据

## 运行方式

### 环境要求

- Python 3.8 或更高版本
- 依赖库：
  - `numpy`：数值计算库
  - `matplotlib`：数据可视化库

### 安装步骤

1. **克隆仓库**

```bash
git clone https://gitee.com/damiao168/cg-lab.git
cd cg-lab
```

2. **安装依赖**

推荐使用 pip 进行安装：

```bash
pip install numpy matplotlib
```

或使用项目提供的配置文件：

```bash
pip install -r requirements.txt
```

3. **启动项目**

```bash
# 使用主入口启动
python main.py

# 或使用src目录下的入口
python src/run.py
```

### 参数配置

运行前可在代码中修改以下参数：

| 参数 | 说明 | 默认值 |
|------|------|--------|
| particle_num | 粒子数量 | 30 |
| max_iter | 最大迭代次数 | 100 |
| w | 惯性权重 | 0.7 |
| c1 | 个体学习因子 | 1.5 |
| c2 | 社会学习因子 | 1.5 |
