# 粒子群算法（PSO）可视化项目

## 项目简介
本项目实现了粒子群优化算法（PSO）的核心逻辑，并结合可视化技术呈现粒子群迭代寻优的交互效果，可直观观察算法的寻优过程与收敛特性。

## 项目架构
├── src/ # 核心代码目录
│ ├── pso_core.py # PSO 算法核心实现（粒子更新、适应度计算、最优解迭代）
│ ├── visualizer.py # 粒子群交互效果可视化
│ └── run.py # 项目启动入口
├── .gitignore # Git 忽略配置文件
└── README.md # 项目说明文档
## 核心逻辑
1. **算法层（pso_core.py）**：定义粒子属性（位置、速度）、适应度计算规则，实现粒子位置/速度更新公式，迭代寻找全局最优解；
2. **可视化层（visualizer.py）**：实时绘制粒子群分布、最优解轨迹，支持交互调整算法参数（粒子数量、迭代次数等）；
3. **执行层（run.py）**：整合算法与可视化模块，初始化参数并启动完整寻优流程。

## 实现功能
- 粒子群算法寻优过程的实时可视化展示；
- 支持自定义目标函数（单峰/多峰）的寻优测试；
- 可调整惯性权重、学习因子等核心参数，观察参数对寻优效果的影响；
- 输出最终寻优结果（全局最优位置、最优适应度值）。
