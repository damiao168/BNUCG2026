# 图标资源说明

请在此目录下放置以下图标文件：

## 所需图标

### TabBar 图标（81px × 81px）

1. `home.png` - 首页图标（未选中状态）
2. `home-active.png` - 首页图标（选中状态）
3. `checkin.png` - 打卡图标（未选中状态）
4. `checkin-active.png` - 打卡图标（选中状态）
5. `mine.png` - 我的图标（未选中状态）
6. `mine-active.png` - 我的图标（选中状态）

## 图标制作建议

- 格式：PNG
- 尺寸：81px × 81px
- 颜色：
  - 未选中：灰色 (#999999)
  - 选中：绿色 (#3cc51f) 或其他主题色
- 背景：透明

## 临时方案

如果暂时没有图标，可以：
1. 从 iconfont、iconpark 等图标库下载
2. 使用在线图标生成工具
3. 暂时注释掉 app.json 中的 tabBar 配置
