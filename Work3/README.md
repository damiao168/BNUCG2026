# 实验四：Phong 光照模型实验

本实验使用 Taichi 实现基于 Ray Casting 的 Phong 光照模型。程序不依赖外部三维模型文件，而是在代码中通过数学隐式方程定义红色球体和紫色圆锥，并对每个像素发射光线进行求交、深度判断和光照计算。
实验中实现了环境光、漫反射和镜面高光三个光照分量，并使用 Taichi GUI 提供交互式滑动条，用于实时调节光照参数，观察不同参数对渲染效果的影响。

## 一、实验目标

1. 理解局部光照模型的基本组成，区分环境光、漫反射和镜面高光。
2. 掌握三维空间中的向量计算，包括法向量、光照方向、视线方向和反射方向。
3. 掌握 Ray Casting 基本流程，实现射线与隐式几何体的求交。
4. 实现类似 Z-buffer 的最近交点深度判断。
5. 使用 Taichi GUI 实现交互式参数调节。
6. 完成选做内容：Blinn-Phong 光照模型和硬阴影效果。

## 二、实验原理

Phong 光照模型将物体表面反射光分为三个部分：

```text
I = I_ambient + I_diffuse + I_specular
````

### 1. 环境光 Ambient

环境光用于模拟场景中经过多次反射后均匀分布的背景光：

```text
I_ambient = Ka × C_light × C_object
```

### 2. 漫反射 Diffuse

漫反射模拟粗糙表面对光的均匀散射，其强度与光线方向和表面法向量夹角有关：

```text
I_diffuse = Kd × max(0, N · L) × C_light × C_object
```

### 3. 镜面高光 Specular

镜面高光模拟光滑表面对光源的强反射效果：

```text
I_specular = Ks × max(0, R · V)^n × C_light
```

其中：

* `N`：表面单位法向量
* `L`：从交点指向光源的单位向量
* `V`：从交点指向摄像机的单位向量
* `R`：理想反射方向
* `n`：高光指数 Shininess

## 三、实现内容

### 1. 场景构建

程序中定义了两个几何体：

* 红色球体

  * 圆心：`(-1.2, -0.2, 0)`
  * 半径：`1.2`
  * 基础颜色：`(0.8, 0.1, 0.1)`

* 紫色圆锥

  * 顶点：`(1.2, 1.2, 0)`
  * 底面高度：`y = -1.4`
  * 底面半径：`1.2`
  * 基础颜色：`(0.6, 0.2, 0.8)`

摄像机和光源设置如下：

* 摄像机位置：`(0, 0, 5)`
* 点光源位置：`(2, 3, 4)`
* 光源颜色：`(1.0, 1.0, 1.0)`

### 2. Ray Casting 光线投射

程序为屏幕上的每一个像素生成一条从摄像机出发的射线，并分别计算该射线与球体、圆锥的交点。

### 3. 深度判断

当一条射线同时击中多个物体时，程序选择距离摄像机最近的正交点进行着色，从而实现类似 Z-buffer 的遮挡判断。

### 4. Phong 着色

在最近交点处计算：

* 世界坐标
* 表面法向量
* 光照方向
* 视线方向
* 反射方向

然后根据 Phong 光照模型计算最终颜色。

### 5. GUI 参数调节

程序使用 `ti.ui.Window` 创建交互窗口，并提供以下滑动条：

| 参数        | 含义     | 范围          | 默认值  |
| --------- | ------ | ----------- | ---- |
| Ka        | 环境光系数  | 0.0 ~ 1.0   | 0.2  |
| Kd        | 漫反射系数  | 0.0 ~ 1.0   | 0.7  |
| Ks        | 镜面高光系数 | 0.0 ~ 1.0   | 0.5  |
| Shininess | 高光指数   | 1.0 ~ 128.0 | 32.0 |

## 实验结果

程序成功渲染出红色球体和紫色圆锥，并能够通过 GUI 实时调节光照参数。调节 `Ka` 可以改变整体环境亮度，调节 `Kd` 可以改变漫反射强度，调节 `Ks` 和 `Shininess` 可以改变高光强度和高光范围。

## 选做内容

### 1. Blinn-Phong 光照模型

在选做代码 `xuanzuo3.py` 中实现了 Blinn-Phong 模型。
Blinn-Phong 使用半程向量 `H` 替代理想反射向量 `R`：

```text
H = normalize(L + V)
I_specular = Ks × max(0, N · H)^n × C_light
```

相比 Phong 模型，Blinn-Phong 的高光计算更加稳定，尤其在大入射角时，高光区域边缘通常更加平滑。

### 2. 硬阴影 Hard Shadow

在交点处向光源方向发射暗影射线。如果暗影射线在到达光源之前击中了其他物体，则认为该点处于阴影中，此时只保留环境光分量。

## 一、操作说明

| 按键    | 功能                  |
| ----- | ------------------- |
| `B`   | 切换 Blinn-Phong 光照模型 |
| `H`   | 切换硬阴影               |
| `Esc` | 退出程序                |

滑动条可实时调节：

* Ka
* Kd
* Ks
* Shininess

## 二、实验结果

按下 `B` 可以切换 Blinn-Phong 光照模型，按下 `H` 可以开启或关闭硬阴影效果。

## 总结

本实验通过 Taichi 实现了基于 Ray Casting 的 Phong 光照模型，完成了球体和圆锥的隐式建模、光线求交、最近深度判断、Phong 着色以及 GUI 参数交互。通过选做内容进一步实现了 Blinn-Phong 模型和硬阴影效果，加深了对局部光照模型、向量计算和实时渲染流程的理解。
