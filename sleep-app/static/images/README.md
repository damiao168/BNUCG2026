# 静态资源说明

请在此目录下放置以下图片资源：

## 目录结构

```
images/
├── audio/              # 音频封面
│   ├── rain.jpg        # 雨声封面
│   ├── ocean.jpg       # 海浪封面
│   ├── forest.jpg      # 森林封面
│   ├── fire.jpg        # 篝火封面
│   ├── white-noise.jpg # 白噪音封面
│   ├── piano.jpg       # 钢琴封面
│   ├── asmr-rain.jpg   # ASMR雨声封面
│   └── story.jpg       # 故事封面
└── default-cover.png   # 默认封面
```

## 图片规格

- 格式：JPG / PNG
- 尺寸：建议 500x500px 或以上
- 大小：单张不超过 200KB
- 风格：柔和、舒缓、符合助眠主题

## 临时方案

如果暂时没有图片资源：
1. 可以使用纯色渐变背景代替
2. 使用在线图片占位服务
3. 先用默认封面，后续替换

## 图片来源建议

1. Unsplash（免费高质量图片）
2. Pexels（免费商用图片）
3. 自己拍摄或设计
